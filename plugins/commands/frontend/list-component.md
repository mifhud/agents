# Find Frontend Features & Components

Provide the frontend features and components available in this application. Focus on elements that help developers understand what's built and how to use it:

## 1. Pages & Routes
- Available routes/pages in the application
- Route parameters and query strings
- Navigation structure

## 2. Reusable Components
- UI components available for reuse
- Component props (required and optional)
- Usage examples

## 3. State Management
- Global state structure
- State management pattern used (Redux, Context, Zustand, etc.)
- How to access and update state

## 4. Forms & Validation
- Form components and validation rules
- How to handle form submission
- Error handling patterns
